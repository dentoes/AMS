Oque é Python?

É uma linguagem de programação simples, muito usada em inteligência artificial, análise de dados, automação e desenvolvimento de sites.

Quais são as principais características do Python?

Sintaxe simples, fácil de aprender, versátil e possui muitas bibliotecas prontas.

Onde o Python é usado?

Em IA, ciência de dados, automação, desenvolvimento web, jogos e aplicativos.

Python é uma linguagem compilada ou interpretada?

Interpetada.

Python é indicado para iniciantes?

Sim, é uma das linguagens mais recomendadas para quem está começando.

JavaScript

Oque é JavaScript?

É uma linguagem usada para criar interatividade em páginas da internet.

Onde o JavaScript é usado?

Sites, aplicações e até aplicativos para celular.

JavaScript funciona apenas no navegador?

Não. Também funciona em servidores.

Qual é a principal função do JavaScript em um site?

Adicionar interatividade, animações e respostas às ações do usuário.

JavaScript é importante para o desenvolvimento web?

Sim, é uma das principais linguagens da área.

C

O que é a linguagem C?

É uma linguagem de programação de baixo nível criada para desenvolver sistemas operacionais e softwares.

Qual é a principal característica da linguagem C?

Alto desempenho e grande controle sobre o hardware.

Onde a linguagem C é usada?

Sistemas operacionais, programas e sistemas embarcados.

A linguagem C é indicada para iniciantes?

Sim, mas exige bastante atenção aos detalhes.

Qual sistema operacional famoso foi desenvolvido em C?

Grande parte do Linux.
